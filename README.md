# renew-k8s-certs

Update k8s cert using ansible-playbook for https://github.com/microsoft/pai

# Usage

Run following command on dev-box machine:

```
git clone https://github.com/siaimes/renew-k8s-certs.git
cd renew-k8s-certs
bash renew-certs.sh
```
